# bubbleSort
**Best Case**
The best case scenario is a sorted list.
(smallest element, 2nd smallest element... ,largest element)
Ex. (1,2,3,4)

**Worst Case**
The worst case scenario is a list in reverse order.
(largest element, second largest element... ,smallest element)
Ex. (4,3,2,1)
